#  Athabasca Oil Sands Water Chemistry Analysis

##  Overview

This repository contains the work completed for a data analysis project examining water chemistry in lakes across the Athabasca oil sands region. The goal was to explore how proximity to industrial activity (Near, Mid, Far) relates to differences in water quality.

---

##  Objectives

* Analyze variation in water chemistry across lake distance groups
* Identify key chemical variables influencing differences between lakes
* Evaluate whether water chemistry can be used to classify lake proximity

---

##  Dataset

Data was compiled from:

* Alberta Government. (n.d.). Oil sands monitoring data portal: Water data station overview. Government of Alberta. https://osmdataportal.alberta.ca/applications/public.html?publicuser=Guest#waterdata/stationoverview
* Government of Alberta. (n.d.). Water quality data portal. https://environment.extranet.gov.ab.ca/apps/WaterQuality/dataportal/

The dataset includes:

* Trace metals (e.g., aluminum, copper, nickel, lead)
* Nutrients (e.g., phosphorus, nitrogen)
* Major ions (e.g., calcium)
* Physical indicators (e.g., pH, dissolved oxygen, turbidity)

---

##  Methods Used

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Principal Component Analysis (PCA)
* Analysis of Variance (ANOVA)
* Random Forest classification
